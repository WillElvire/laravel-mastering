Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

GET api/user

Example request:
curl --request GET \
    --get "http://localhost/api/user" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/user"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

POST api/v1/auth/register

Example request:
curl --request POST \
    "http://localhost/api/v1/auth/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"firstname\": \"bospjhobkfoyarkxcxtcnj\",
    \"lastname\": \"sj\",
    \"email\": \"mohammad11@example.org\",
    \"password\": \"\\\"%A&3*+`G@3BfJJO\"
}"

const url = new URL(
    "http://localhost/api/v1/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "firstname": "bospjhobkfoyarkxcxtcnj",
    "lastname": "sj",
    "email": "mohammad11@example.org",
    "password": "\"%A&3*+`G@3BfJJO"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/v1/auth/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

firstname   string   

Must not be greater than 255 characters. Example: bospjhobkfoyarkxcxtcnj

lastname   string   

Must not be greater than 255 characters. Example: sj

email   string   

Must be a valid email address. Must not be greater than 255 characters. Example: mohammad11@example.org

password   string   

Must be at least 8 characters. Example: "%A&3*+G@3BfJJO`
